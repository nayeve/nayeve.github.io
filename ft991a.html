Setup Notes

  • FT-991A: Configure for split mode (VFO-A uplink, VFO-B downlink). Set CAT rate to 9600 baud (Menu 031).
  • Gpredict: Install from Gpredict Site. Update TLEs, set FT-991A as rig, GS-232B for rotor control.
  • G-5500DC: Mount on mast or tower. Use GS-232B for computer control. Two 5-conductor cables required (AZ and EL).
  • Antenna: Arrow II Yagi is handheld or mast-mountable. Align for correct polarization (RHCP for most sats).
  • Operation: Use Gpredict to track satellites, adjust Doppler, and point the Yagi. Start with FM sats like SO-50 for easy QSOs.
